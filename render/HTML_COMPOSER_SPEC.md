# HTML Composer Specification

## Template Engine
Use Jinja2 (Python) or React SSR (JS).

## Layout
* **Header**: Framing context.
* **Body**: Sections based on `ScenarioJudgementSheet` schema.
* **Interactivity**: 
    * No JS frameworks required for viewership (static HTML).
    * "Evidence Cards" implemented as `<details>` or simple modals.
