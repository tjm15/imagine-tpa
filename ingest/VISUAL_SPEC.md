# Visual Understanding Specification

## Model Choice
* **Azure**: Azure OpenAI (GPT-4o).
* **OSS**: Qwen3-VL.

## Tasks
1. **Plan Reading**: "Where is the north arrow? What is the scale?"
2. **Site Assessment**: "Is this site overgrown? Is it flat?"
3. **Chart Reading**: Extract data points from embedded charts.

## Output
All VLM outputs must be structured (JSON), not chat.
